Design robust mastery learning curriculum using Skycak's framework:
Core Principles:
Solve Bloom's Two-Sigma Problem via technology+cognitive science
Strict mastery requirement: complete understanding before advancement
4x learning acceleration through systematic cognitive strategies
Architecture Components:
Knowledge Graph: Map hierarchical prerequisites+"encompassings" where advanced practice reinforces foundations
Spaced Repetition: Hierarchical algorithm with trickle-down effects - advanced topics provide partial review credit for prerequisites
Cognitive Load Management: Worked examplesscaffolded practicemastery demonstration (2 correct consecutive)
Implementation Framework:
Expert system (not ML) emulating tutor decisions
Real-time optimization: what learn next/review/sequence
Continuous measurement+accountability
Diagnostic placementmastery criteriareview scheduling
Design Sequence:
Domain analysis: complete knowledge structure mapping
Define measurable mastery criteria per level
Create worked examples for each concept
Implement hierarchical spaced repetition
Trick performance data for optimization
Key Principles:
Fortify fundamentals to "comfortable fluency"
Embrace productive struggle ("intense workouts")
Address computational errors as conceptual indicators
Measure+improve continuously
Application: Adaptable to any hierarchical knowledge domain. Prioritize mastery over coverage, use technology for algorithmic decision-making about optimal learning paths based on individual performance patterns.
